Clearly

Clearly is a programmable structured text editor. If you know what it means you can go and fork its GitHub repository. Otherwise, read tutorial ↓ and start using it :)

It is structured - every block of text is wrapped inside a paragraph. Paragraphs are grouped inside sections:

Navigation

The editor is a webpage. You can scroll it with scrollbar and mouse wheel.

One HTML element is always selected - its left border is slightly thicker and darker.

When you click on an element it also gets activated. Watch out not to click it twice though - it would start edit mode. You can leave edit mode with [Escape] or by clicking outside currently edited element.

You can use arrows to navigate ([←] [→] [↑] [↓]) between next/previous elements and to ascend/descent in their hierarchy.

You navigate inside the webpage as it was a tree. Hold down [Ctrl] and press up/down arrows to walk across elements at different levels.

[Home] and [End] keys will move you to the first and last element in the current section. Holding down [Shift] will cause you to move to start and end of the document.

Structure Manipulation

Select an element and press one of the digit keys from [1] to [6]. You can create headers this way.

If you want to turn a header back to a paragraph, press the [p] key while the header is selected.

Press [Delete] with an element selected to remove it from the page. Elements removed this way are not lost - they are put into killring instead.

You can retrieve an element from killring and paste it after current element by pressing [Insert]. Press [Shift + Insert] to add it before.

If you press [Ctrl + \] you will insert new horizontal rule after selected element. Hold Shift to add it before instead.

You can hide elements by pressing the [h] key. Pressing it again will show the element back. It allows you to compact sections that are not important.

Press [Ctrl + Shift + →] to wrap current element inside a section.

Similarily [Ctrl + Shift + ←] will cause the element to unwrap a section.

Holding down [Shift] while pressing [↑] and [↓] will cause current element to bubble. You can supply this command with [Ctrl] to bubble accross other element boundaries.

TODO Lists

Clearly allows you to store and manage multiple task lists by annotating elements with task states (todo, cancelled, done, etc.).

Press [Tab] to change task state of the active element.

You can press it a few times - it cycles all possible states.

[Shift + Tab] cycles todo states backwards.

Text Editing

Press [Enter] anytime to start editing active element.

If you press [Ctrl + Enter], new element will be created before edit mode starts.

To exit edit mode press [Escape] or click on an other elment.

Pressing [Enter] while in edit mode will create new paragraph. Empty paragraphs are automatically destroyed.

Text Formatting

You can format selected text by pressing [Ctrl + _], where _ is one of:

  • [Escape] - Remove formatting.
  • [b] - Bold text.
  • [i] - Italic text.
  • [u] - Underline.
  • [s] - Strike through.
  • [↑] - Superscript.
  • [↓] - Subscript.
  • [e] - Make link from URL.
  • [-] - Decrease font size.
  • [+] - Increase font size.
